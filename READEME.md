\# CVE Vulnerability Scanner



A Python-based Vulnerability Scanner that fetches known CVEs from the NVD (National Vulnerability Database).



\## Features

\- Fetches real-world CVEs

\- Uses official NVD API

\- SOC Analyst friendly

\- Beginner level project



\## Requirements

\- Python 3.x

\- requests library



\## Installation

pip install -r requirements.txt



\## Usage

python cve\_scanner.py
